## File Format Information

### EML Format (.eml)
- **Standard:** RFC 822 (Internet Message Format)
- **Compatibility:** Universal email format
- **Readability:** Can be opened in any email client or text editor
- **Forensic Value:** Contains complete email headers, metadata, and content
- **Conversion:** Successfully converted to Markdown + PDF

### MSG Format (.msg)
- **Standard:** Microsoft Outlook proprietary format
- **Compatibility:** Requires Microsoft Outlook or specialized tools
- **Readability:** Best viewed in Microsoft Outlook
- **Forensic Value:** Contains complete email metadata and attachments
- **Conversion:** Converted via .eml equivalents (all .msg files have .eml versions)
